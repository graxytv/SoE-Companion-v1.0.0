# SoE Companion Changelog

## 1.0.0

- Initial public release of SoE Companion for Sanctuary of Exile single player.
- Added a Diablo-inspired app redesign with Home, General, Drops Tracker, Loot Filter, Notifications, Sounds, Achievements, Holy Grail, and SoE Wiki tabs.
- Added selectable themes: Sanctuary Gothic, Hellfire, Horadric Parchment, Classic Dark, and Classic Light.
- Added a Home dashboard with launcher support, total kills, achievements, Holy Grail progress, version info, creator links, donate link, and setup wizard entry.
- Added Holy Grail tracking for uniques, Hellforged uniques, set items, runewords, and all runes.
- Added clickable Holy Grail items with item detail panels where SoE wiki data is available.
- Added the Runeword planner, which reads rune counts from the shared stash Materials tab and shows makeable or missing runewords.
- Added optional Auto Grail Tracker support through the bundled drop hook and `C:\grail_drops.log`.
- Added identified item tracking for new grail entries, including items identified from inventory.
- Added Drop Tracker and Total Drops tracking with session/run timers, recent tracked items, duplicate prevention, reset controls, and SoE-specific categories.
- Added rune tracking by tier: Low Rune, Mid Rune, and High Rune, plus per-rune counts.
- Added in-game overlays for notifications, Drops Tracker, Total Drops, Holy Grail progress, Rune Tracker, Muling Mode, and achievement unlocks.
- Added overlay position and size controls, an optional separate tracker window, and an Always Show Overlays setting.
- Added an account-wide Achievements tab with progress bars, unlock popups, backups, reset controls, and sync support for account stats.
- Added achievement categories for unique finds, kills, bossing, levels, grail progress, runes, materials, and corruptions.
- Added material test achievements for SoE materials, orbs, sigils, boss materials, and perfect gems.
- Added custom sound slots, sound testing, volume controls, achievement sounds, downloadable sound packs, and item-specific notification sounds.
- Added per-item sound rules for uniques, Hellforged items, sets, runes, materials, and custom item names.
- Added a native SoE loot filter editor with bundled Hiim_SOE and Kryzard_SoE presets, filter management, copy tools, and install instructions.
- Added New Game Automation settings and hotkey support.
- Added local backup and restore support for Holy Grail and Achievements.
- Disabled D2MXLUtils update checks and removed legacy MXL-only UI features from the active app.
